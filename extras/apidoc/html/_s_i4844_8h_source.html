161 /*
162  * English:
163  * Handling interruptions.
164  * Whenever the status of the ATDD changes, a hardware interrupt is triggered. For example, when you move the tuner
165  * potenciometer, the ATDD sends a signal to the Arduino pin (INTERRUPT_PIN). The same is true when the ATDD is capturing
166  * mono FM signal and has switched to stereo. The variable below indicates a change of the ATDD status. It will need to
167  * process some action (for example show on LCD this changes).
168  *
169  * Portuguese:
170  * Manipulando Interrupções
171  * Sempre que o estado do ATDD é modificado, uma interrupção de hardware é disparada. Esta interrupção é manipulada por
172  * uma função implementada neste sketch, que nada mais faz que dizer que isso ocorreu para que algumas ações seja executadas adiante.
173  * Por exemplo: Qundo potenciômetro do sintonizador for girado, o ATDD envia um sinal para o pino do Arduino, no nosso caso, o
174  * pino 2 do mini pro (INTERRUPT_PIN). Da mesma forma, quando um sinal de FM está sendo capturado no modo MONO e o sinal, por
175  * alguma razão, melhora e passa para FM, esta interrupção ocorre indicando que devemos apresentar FM Estéreo.
176  *
177  */
